# Sabbir95 Android Build Guide

আপনার এই প্রজেক্টটিকে অ্যান্ড্রয়েড অ্যাপ (APK) হিসেবে তৈরি করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

## ১. প্রয়োজনীয় সফটওয়্যার
আপনার কম্পিউটারে নিচের সফটওয়্যারগুলো থাকতে হবে:
- **Node.js** (ইন্সটল করা আছে)
- **Android Studio** (ডাউনলোড লিংক: https://developer.android.com/studio)

## ২. অ্যাপ আপডেট করার প্রক্রিয়া (যখনই কোড পরিবর্তন করবেন)
আপনি যখনই এই প্রজেক্টে কোনো পরিবর্তন করবেন, অ্যান্ড্রয়েড অ্যাপে সেই পরিবর্তনগুলো নিতে নিচের কমান্ডগুলো দিন:

```bash
# ১. প্রথমে প্রজেক্ট বিল্ড করুন
npm run build

# ২. পরিবর্তনগুলো অ্যান্ড্রয়েড ফোল্ডারে সিঙ্ক করুন
npx cap sync
```

## ৩. APK তৈরি করার প্রক্রিয়া
১. আপনার কম্পিউটারে **Android Studio** ওপেন করুন।
২. **"Open an Existing Project"** এ ক্লিক করে আপনার প্রজেক্টের ভেতরের `android` ফোল্ডারটি সিলেক্ট করুন।
৩. অ্যান্ড্রয়েড স্টুডিও সব ফাইল লোড করলে, উপরে মেনু থেকে **Build > Build Bundle(s) / APK(s) > Build APK(s)** এ ক্লিক করুন।
৪. বিল্ড শেষ হলে একটি পপআপ আসবে, সেখানে **"locate"** এ ক্লিক করলে আপনি আপনার `.apk` ফাইলটি পেয়ে যাবেন।

## ৪. ভবিষ্যতে মডিফাই করা
আপনি যদি ভবিষ্যতে আরও কিছু পরিবর্তন করতে চান (যেমন নতুন চ্যানেল যোগ করা), তবে আপনি এই এডিটরেই কোড পরিবর্তন করবেন। পরিবর্তন শেষ হলে উপরের **ধাপ ২** অনুসরণ করে সিঙ্ক করবেন এবং অ্যান্ড্রয়েড স্টুডিও থেকে নতুন APK তৈরি করবেন।

## ৫. গুরুত্বপূর্ণ: HTTP চ্যানেল চালানোর জন্য (Cleartext Traffic)
আপনার অ্যাপে যেহেতু কিছু HTTP চ্যানেল আছে (যেমন PTV Sports, Willow TV), তাই অ্যান্ড্রয়েড অ্যাপে এগুলো চালানোর জন্য একটি ছোট কনফিগারেশন পরিবর্তন করতে হবে।

১. **Android Studio** তে প্রজেক্ট ওপেন করুন।
২. বাম পাশের ফাইল এক্সপ্লোরার থেকে `app > src > main > AndroidManifest.xml` ফাইলটি ওপেন করুন।
৩. `<application` ট্যাগের ভেতরে নিচের লাইনটি যোগ করুন (যদি না থাকে):

```xml
android:usesCleartextTraffic="true"
```

উদাহরণস্বরূপ:
```xml
<application
    android:allowBackup="true"
    android:icon="@mipmap/ic_launcher"
    android:label="@string/app_name"
    android:roundIcon="@mipmap/ic_launcher_round"
    android:supportsRtl="true"
    android:theme="@style/AppTheme"
    android:usesCleartextTraffic="true">  <!-- এই লাইনটি যোগ করুন -->
```

এটি না করলে অ্যান্ড্রয়েড অ্যাপে HTTP চ্যানেলগুলো চলবে না।
